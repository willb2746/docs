# Phonely API Reference

The Phonely API provides a powerful interface for generating fast completions using Phonely's language models.

## Base URL

```
https://fast.phonely.ai/openai/v1
```

## Authentication

All API requests require authentication. You need to include your API key in the Authorization header:

```bash
Authorization: Bearer $PHONELY_API_KEY
```

You can obtain an API key from the [Phonely Dashboard](https://dashboard.phonely.ai).

## Available Endpoints

The API reference provides details on the following endpoints:

- [Fast Completions](https://phonely.ai/api-reference/chat/create-response) - Create fast completions with Phonely's language models
